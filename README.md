# Linux Basics DevOps Lab

This is my first DevOps learning lab.

## What this project contains

- Basic Linux folder structure
- Bash script for system information
- Git version control practice

## Script

The script `scripts/system-info.sh` shows:

- Current user
- Current date
- Current directory
- Linux kernel version
- Disk usage
- Memory usage

## How to run

```bash
chmod +x scripts/system-info.sh
./scripts/system-info.sh
